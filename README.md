# Blauwtrust

# Installation

Install blauwtrust with composer

```bash
  composer install 
```

To execute Export Pay Salaries Data Command , make sure you are on the main directory and run :  

```bash
  php bin/console  app:export-pay-salaries-data blauwtrust-test
```

To run PHPUnit tests run :  

```bash
  php bin/phpunit
```
